# 现代志怪（新志怪）阅读整理

- 来源：用户提供文本（2026-04-24）
- 说明：以下内容按用户原文整理，暂未另行核验。

现代志怪（也称“新志怪”）非常活跃，分**严肃文学、都市怪谈、网络志怪**三类，核心是把《聊斋》《搜神记》传统放进**当代都市/现代生活**，写精怪、异闻、人性寓言。

### 一、严肃文学派（名家，偏文学性）
1. **燕垒生《神州志异·燕垒怪谈》**（2026）
   当代文言志怪代表，短故事集，传统精怪（狐、鬼、器灵）+现代市井，文笔古雅、意境清冷。

2. **李碧华《饺子》《胭脂扣》**
   香港都市志怪巅峰，日常（茶楼、厨房、菜市场）藏鬼魅，写情欲、贪婪与执念，文字妖异入骨。

3. **王蒙《极限聊斋》**（2025）
   先锋重写《聊斋》（聂小倩、画皮等），意识流+现代心理，古典狐鬼住进当代人的时间与记忆。

4. **宋方金《两心》**（2025）
   改编《搜神记·秦巨伯斗鬼》，现代版人鬼对峙，寓言人性与猜忌，水墨风封面。

### 二、都市怪谈/单元剧（现代都市+精怪，好读）
1. **《中国异闻录》**（40+单元）
   现代版《聊斋》，主角带玉珠卷入异闻：人皮鼓、鬼新娘、灵车司机、风水秘术，都市悬疑+中式恐怖。

2. **《守夜人的精怪簿》**
   起点都市志怪，单元剧，雪珠、祸斗、貔貅等传统精怪现代表演，藏在写字楼、地铁、便利店，写欲望与救赎。

3. **《聊斋奇谈：当代50篇》**
   现代都市怪谈：地铁狐仙、外卖黄鼠狼、健身房山魈、高架桥龙影，50个独立故事，刺现代人性。

### 三、网络新志怪（设定新、脑洞大）
1. **《从民俗论坛开始，编造镇物》**
   诡异复苏世界观，主角编民俗谣言，信的人多就成真，从论坛编到“地府炼假成真”，民俗+幕后流。

2. **《规则类怪谈：4016》**
   校园规则怪谈，守则、告示牌、异常天气，逻辑闭环，克苏鲁风+中式诡异，无超自然硬设定。

### 四、现代志怪的特点（和《聊斋》区别）
- **背景**：古代荒村 -> **现代都市/互联网/校园/职场**
- **精怪**：传统狐鬼 -> **数据幽灵、二维码索命、外卖精怪、写字楼灵**
- **内核**：科举/礼教讽刺 -> **内卷、孤独、消费主义、科技异化**
- **风格**：文言/白话 -> **口语化、碎片化、单元剧、强悬疑**

### 五、入门阅读推荐
- 想读**文笔好、有文学感**：燕垒生《神州志异》、李碧华《饺子》
- 想读**好读、都市悬疑**：《中国异闻录》《守夜人的精怪簿》
- 想读**脑洞设定**：《规则类怪谈：4016》
