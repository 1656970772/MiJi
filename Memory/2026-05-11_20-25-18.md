## 外部来源核验

补充核验来源：

- Epic 官方 Forward Shading Renderer 文档说明，Unreal Engine 默认使用 Deferred Renderer，因为它更通用并能访问更多渲染特性。
- Epic 官方 RDG 文档说明，Rendering Dependency Graph 是基于图的调度系统，用于对整帧渲染管线做优化；它会收集 pass，再编译并按依赖顺序执行。
- Epic 官方移动渲染文档说明，UE 为移动端提供 Forward 与 Deferred shading mode。
- Epic 官方 Rendering Project Settings 文档说明，`Clear Scene` 选项定义 game mode 下 GBuffer 如何清除，并注明只影响 deferred shading。
