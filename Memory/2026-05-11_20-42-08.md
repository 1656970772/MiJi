## 外部来源核验

补充核验来源：

- Epic 官方 Mesh Drawing Pipeline 文档说明，`FPrimitiveSceneProxy` 通过 `GetDynamicMeshElements` 和 `DrawStaticElements` 回调向 renderer 提交 `FMeshBatch`。
- Epic 官方 Mesh Drawing Pipeline 文档说明，`FMeshBatch` 用于将 `FPrimitiveSceneProxy` 实现和 mesh pass 解耦。
- Epic 官方 Mesh Drawing Pipeline 文档说明，`FMeshDrawCommand` 是 `FMeshBatch` 和 RHI 之间的接口，包含 RHI 对 mesh draw 需要知道的信息。
- Epic 官方 Mesh Drawing Pipeline 文档说明，`FMeshDrawCommand` 由 mesh pass 特定的 `FMeshPassProcessor` 从 `FMeshBatch` 创建，最后通过 `SubmitMeshDrawCommands` 转为 RHI command。
- Epic 官方 `FMeshBatch` API 文档说明，`FMeshBatch` 是一批具有相同 material 和 vertex buffer 的 mesh element。
- Epic 官方 `FMeshDrawCommand::SubmitDraw` API 文档说明，该函数向 RHI CommandList 提交 MeshDrawCommand 的绘制命令。
