## 外部来源核验

补充核验来源：

- Microsoft Learn PIX 概览说明 PIX 面向 Direct3D 12 游戏开发者，可用 GPU Captures 调试渲染问题和分析帧性能，也可用 Timing Captures 做 profiling。
- Microsoft PIX 官方 GPU Captures 文档说明 GPU Capture 会记录 Direct3D 12 API 调用和参数，之后 replay 以启用调试和分析；首次打开捕获后需要 Start Analysis 才能完整使用功能。
- Microsoft PIX GPU Captures 文档说明 Collect Timing Data 会 replay 捕获的 API 调用并测量 GPU 执行时间。
- Microsoft Download Center 提供 PIX on Windows 安装包，支持 Windows 10/11。
- Microsoft Learn Developer Mode 文档说明 Developer Mode 面向构建、部署、测试应用，启用需要管理员权限。
- Unreal 官方文档说明 `r.ShaderDevelopmentMode=1` 可在 `ConsoleVariables.ini` 中获取 shader 编译详细日志。
- Unreal Shader Debugging Workflows 文档说明 UE5 中 `r.Shaders.KeepDebugInfo` 被拆分为 `r.Shaders.Symbols` 与 `r.Shaders.ExtraData`。
